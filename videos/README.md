# 영상 파일 가이드

## 📁 영상 파일 위치
이 폴더에 히어로 섹션에서 사용할 영상 파일을 넣어주세요.

## 🎬 권장 영상 파일

### 파일명
- `hero-video.mp4` (메인 영상 파일)
- `hero-video.webm` (선택사항 - 더 나은 압축률)

### 영상 사양
- **해상도**: 1920x1080 (Full HD) 이상 권장
- **비율**: 16:9 (가로형)
- **길이**: 10-30초 (반복 재생)
- **파일 크기**: 5MB 이하 권장 (로딩 속도 고려)
- **포맷**: MP4 (H.264 코덱) 또는 WebM

### 영상 내용 추천
- 펜션, 카페, 카라반, 캠핑장 인테리어 관련
- 아늑하고 따뜻한 분위기
- 고품질의 인테리어 공간
- 자연스러운 조명과 색감

## 🔧 영상 최적화 팁

### 파일 크기 줄이기
1. **해상도 조정**: 1920x1080 → 1280x720
2. **비트레이트 조정**: 2-5 Mbps
3. **프레임레이트**: 24-30fps
4. **압축 설정**: 중간 품질

### 권장 도구
- **FFmpeg** (명령줄)
- **HandBrake** (GUI)
- **Adobe Media Encoder**
- **온라인 압축 도구**

## 📱 호환성
- **MP4**: 모든 모던 브라우저 지원
- **WebM**: Chrome, Firefox, Edge 지원
- **대체 이미지**: 영상 로딩 실패 시 자동 표시

## ⚠️ 주의사항
1. 저작권이 있는 영상 사용 금지
2. 파일명은 정확히 `hero-video.mp4`로 설정
3. 영상 파일이 없으면 자동으로 대체 이미지 표시
4. 자동 재생이 차단된 경우 재생 버튼 표시

## 🚀 사용 방법
1. 영상 파일을 이 폴더에 `hero-video.mp4` 이름으로 저장
2. 웹사이트 새로고침
3. 히어로 섹션에서 영상 재생 확인
